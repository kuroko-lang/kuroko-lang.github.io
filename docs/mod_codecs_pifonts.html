_class Cp042IncrementalDecoder(IncrementalDecoder)

Decoder for Windows code page 42 (GDI Symbol), and base class for symbol font decoders.

This maps characters to PUA with the low 8 bits matching the original byte encoding, similarly to x-user-defined, but using a different PUA range and including all non-C0 bytes, not only non-ASCII bytes.
