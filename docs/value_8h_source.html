35 /*
36  * The following poorly-named macros define bit patterns for identifying
37  * various boxed types.
38  *
39  * Boxing is done by first setting all of the bits of MASK_NAN. If all of
40  * these bits are set, a value is not a float. If any of them are not set,
41  * then a value is a float - and possibly a real NaN.
42  *
43  * Three other bits - one before and two after the MASK_NAN bits - determine
44  * what type the value actually is. KWARGS sets none of the identifying bits,
45  * NONE sets all of them.
46  */
